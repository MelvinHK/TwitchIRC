# Twitch Chat Relay Web App

Relays the chatroom of a specified channel name.

## Setup:
1. npm init
2. npm start
3. open browser and navigate to http://localhost:4000/
4. input valid channel name in the relay search bar

## Features:
- Channel name search
- Message filters
    - By username(s)
    - Include/Exclude string(s)
- Pause button
